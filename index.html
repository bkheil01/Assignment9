This page explains the steps I took to create my Dog Lover's Blog.

First I created an MVC.net project in Visual Studio. Visual studio created everything that I would need to create this blog, including bootstrap.

Second I created a Blog Model that contained Name, About, multiple Past fields, Image, Favorite, Last, Date. This is for the blogger's Name, About the blogger's dog, multiple past fields for previously adopted dogs, an Image field for the owner's dog, favorite breed of dog, Last dog adopted, and the date of the last adoption.

Third I created a view to set all of the fields above on the page in the order that I wanted them.

Finally I create a controller called blogcontroller that will push all of the fields from the view to the model and vice versa.
